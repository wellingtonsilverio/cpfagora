# cpfagora

## Install and Run
.env: file with the settings (required)
```
ENV = 
PORT = 

MONGO_USER = 
MONGO_PASS = 
MONGO_URI = 

CPFCNPJ_API = 
RECEITAWS_API = 
```

### Clone repo and enter in it
```bash
git clone https://github.com/wellingtonsilverio/cpfagora.git
cd cpfagora
```
### Install nodejs modules
```bash
npm i
```
### Run developement mode
```bash
npm run serve
```
